css animations

apa itu CSS animations?

Animasi memungkinkan elemen berubah secara bertahap dari satu gaya ke gaya lainnya. Anda dapat mengubah sebanyak mungkin properti CSS yang anda inginkan, untuk menggunakan CSS animations, anda harus menentukan beberapa keyframes untuk animasi. keyframes memegang gaya apa yang akan dimiliki elemen pada waktu-waktu tertentu.

CSS animation-nama dan animation-duration

Properti animation-name menentukan nama untuk animasi. Sedangkan, Properti animation-duration mendefinisikan berapa lama animasi harus dilakukan. Jika properti ini tidak ditentukan, maka tidak ada animasi yang terjadi, karena nilai default adalah 0s(0 detik).

CSS aturan @keyframes

Ketika anda menentukan gaya CSS di dalam @keyframes, animasi secara bertahap akan berubah dari gaya saat ini(default) ke gaya baru pada waktu-waktu tertentu. Untuk mendapatkan animasi yang bekerja, anda harus mengikat animasi ke elemen.
Contoh beriut memanggil animasi animation1, 2, dan 3 ke class aturan1, 2, dan 3. Animasi akan berlangsung selama 5s(5 detik), dan secara bertahap akan mengubah background-color dari class aturan1, 2, dan 3 dari red ke green.
