css animations

apa itu CSS animations?

Animasi memungkinkan elemen berubah secara bertahap dari satu gaya ke gaya lainnya. Anda dapat mengubah sebanyak mungkin properti CSS yang anda inginkan, untuk menggunakan CSS animations, anda harus menentukan beberapa keyframes untuk animasi. keyframes memegang gaya apa yang akan dimiliki elemen pada waktu-waktu tertentu.

CSS animation-nama dan animation-duration

Properti animation-name menentukan nama untuk animasi. Sedangkan, Properti animation-duration mendefinisikan berapa lama animasi harus dilakukan. Jika properti ini tidak ditentukan, maka tidak ada animasi yang terjadi, karena nilai default adalah 0s(0 detik).

CSS aturan @keyframes

Ketika anda menentukan gaya CSS di dalam @keyframes, animasi secara bertahap akan berubah dari gaya saat ini(default) ke gaya baru pada waktu-waktu tertentu. Untuk mendapatkan animasi yang bekerja, anda harus mengikat animasi ke elemen.

CSS properti animation-direction

Properti animation-direction menentukan apakah animasi harus dimainkan dengan nilai normal, reverse, alternate, alternate-reverse.
Properti animasi-direction dapat memeiliki nilai-nilai berikut:

  • normal - Animasi dimainkan seperti biasa(default).
  • reverse - Animasi dimainkan mulai dari arah mundur.
  • alternate - Animasi dimainkan denagn awal ke depan, kemudian mundur.
  • alternate-reverse - Animasi dimainkan dengan awal mundur, kemudian ke depan.

Berikut adalah contoh untuk masing-masing nilai animation-direction menggunakan animasi yang sama namun dengan arah yang berbeda.

CSS properti animation-timing-function

Properti animation-timing-function menentukan kurva kecepatan animasi. Kurva kecepatan mendefinisikan bagaimana animasi berubah dari satu set nilai ke set nilai lainnya.
Properti animation-timing-function dapat memiliki nilai-nilai berikut:

  • ease - Animasi dimulai lambat, kemudian cepat, dan berakhir lambat (default).
  • linear - Animasi memiliki kecepatan yang sama dari awal hingga akhir.
  • ease-in - Animasi dimulai lambat.
  • ease-out - Animasi berakhir lambat.
  • ease-in-out - Animasi dimulai dan berakhir lambat.
  • step-start - Animasi melompat langsung ke nilai akhir pada awal animasi.
  • step-end - Animasi tetap pada nilai awal hingga akhir animasi, lalu melompat ke nilai akhir.
  • steps(n, start|end) - Menentukan langkah-langkah diskrit dalam animasi.
  • cubic-bezier(n,n,n,n) - Memungkinkan definisi kurva kecepatan kustom.

Berikut adalah contoh untuk masing-masing nilai animation-timing-function menggunakan animasi yang sama namun dengan kurva kecepatan yang berbeda.
