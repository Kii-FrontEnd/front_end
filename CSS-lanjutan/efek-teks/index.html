Text Effect

CSS memiliki beberapa properti untuk menangani teks overflow, word-wrap, word break, dan writing mode. Dalam bab ini anda akan belajar tentang properti berikut:

  • text-overflow - menentukan cara menangani kontent yang meluap.
  • word-wrap - Memungkinkan kata-kata panjang untuk dapat dipatahkan dan dibungkus ke baris berikutnya.
  • word-break - Menentukan aturan melanggar garis.
  • writing-mode - Menentukan apakah baris teks diletakkan secara horizontal atau vertikal.

CSS text-overflow properti

CSS properti text-overflow menentukan bagaimana konten yang meluap dari elemen ditampilkan kepada pengguna. Properti ini hanya berfungsi pada elemen yang memiliki overflow: hidden dan white-space: nowrap. Nilai yang dapat digunakan adalah:

Nilai-nilai text-overflow:

  • clip - Teks yang meluap akan dipotong dan tidak terlihat (nilai default)
  • ellipsis - Teks yang meluap akan diganti dengan "..." (tiga titik)
  • string - Teks yang meluap akan diganti dengan string yang ditentukan pengguna

CSS word-wrap properti

CSS properti word-wrap memungkinkan kata-kata panjang yang meluap dari batas elemen untuk dapat dipatahkan dan dibungkus ke baris berikutnya. Properti ini sangat berguna ketika anda memiliki teks yang sangat panjang tanpa spasi. Nilai yang dapat digunakan adalah:

Nilai-nilai word-wrap:

  • normal - Hanya memotong di titik pemisah yang diperbolehkan (nilai default)
  • break-word - Memungkinkan kata panjang untuk dipatahkan dan dibungkus ke baris berikutnya

CSS word-break properti

CSS properti word-break menentukan aturan bagaimana teks harus dipatahkan ketika mencapai akhir dari sebuah elemen. Properti ini berbeda dengan word-wrap karena properti ini akan mempatahkan kata bahkan di tengah-tengah kata. Nilai yang dapat digunakan adalah:

Nilai-nilai word-break:

  • normal - Menggunakan aturan pemecahan baris default (nilai default)
  • break-all - Memungkinkan kata dipatahkan di mana saja untuk mencegah overflow
  • keep-all - Mencegah pemecahan kata pada bahasa CJK (Chinese, Japanese, Korean)

CSS writing-mode properti

CSS properti writing-mode menentukan apakah baris teks diletakkan secara horizontal atau vertikal. Properti ini sangat berguna untuk menampilkan teks dalam berbagai arah, terutama untuk bahasa-bahasa seperti Cina, Jepang, dan Korea. Nilai yang dapat digunakan adalah:

Nilai-nilai writing-mode:

  • horizontal-tb - Teks mengalir dari kiri ke kanan, baris dari atas ke bawah (nilai default)
  • vertical-rl - Teks mengalir dari atas ke bawah, baris dari kanan ke kiri
  • vertical-lr - Teks mengalir dari atas ke bawah, baris dari kiri ke kanan
  • sideways-rl - Teks miring ke kanan, baris dari kanan ke kiri
  • sideways-lr - Teks miring ke kiri, baris dari kiri ke kanan
