CSS color keywords

CSS memiliki beberapa kata kunci warna predefined khusus yang dapat digunakan dengan properti seperti color, background-color, dan border-color.
Kata kunci khusus adalah:

  • transparent
  • currentcolor
  • inherit

Kata kunci transparent

Kata kunci transparent digunakna untuk membuat warna transparant. Hal ini sering digunakan untuk membuat warna latar belakang transparan untuk sebuah elemen.

Catatan: transparent kata kunci setara dengan rgba(0, 0, 0, 0). Nilai warna RGBA adalah perpanjangan dari nilai warna RGB dengan tambahan alfa(opacity) - yang menentukan opacity untuk warna.

Kata kunci currentcolor

Kata kunci currentcolor itu seperti variabel yang memegang nilai saat ini dari properti warna elemen.
Kata kunci ini dapat berguna jika anda ingin warna tertentu konsisten dalam elemen atau halaman.
