css linear gradient

CSS fungsi linear-gradient() menciptakan gradien linear.
Gradien linier mendefiniskan transisi warna yang berjalan dalam garis lurus, bisa turun, ke atas, ke kiri, ke kanan atau diagonal.
Sebuah gradien linier membutuhkan setidaknya dua warna berhenti. Color stop adalah warna yang ingin anda buat transisi yang halus. Anda juga dapat menetapkan titik awal dan arah (atau sudut) bersama dengan efek gradien.

Sintaks

background-image: linear-gradient(direction, color-stop1, color-stop2, ....);

arah - top to bottom (ini adalah default)
