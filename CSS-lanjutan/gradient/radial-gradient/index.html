CSS fungsi radial-gradient()

CSS fungsi redial-gradient() menciptakan gradient radial.
Fungsi ini mendefinisikan transisi warna yang keluar dari titik pusat. Gradient radial membtuhkan setidaknyya dua warna.

Sintaks

background-image: radial-gradient(shape size at position, start-color, .... , last-color);

Secara default, shape adalah elips, size adalah tikungan terjauh, dan position adalah pusat.

Radial gradient - set shape

Parameter shape mendefinisikan bentuk gradient. Ini dapat mengambil salah satu nilai berikut:

  • Elisse(ini adalah default)
  • Circle

Contoh berikut menunjukkan radial gradient dengan bentuk lingkaran:

Contoh

nilai elisse
nilai circle

Radial gradien - parameter size

Parameter size mendefiniskan size gradien bentuk. Ini dapat mengambil salah satu nilai berikut:

  • closest-side
  • farthest-side
  • closest-corner
  • farthest-corner(ini default)
