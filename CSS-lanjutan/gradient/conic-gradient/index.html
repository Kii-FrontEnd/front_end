CSS conic-gradient() fungsi

Fungsi CSS conic-gradient() menciptakan gradien yang kerucut. Fungsi ini merupakan gradien dengan transisi warna yang memutar di sekitar titik pusat. Untuk membuat gradien yang unik, anda harus menentukan setidaknya dua warna.

Sintaks

background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...);

Secara default, angle adalah 0deg dan position adalah pusat. Jika tidak degree ditentukan, warna akan tersebar sama-sama titik tengah.
