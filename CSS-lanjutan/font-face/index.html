Apa itu @font-face?

@font-face adalah aturan CSS yang memungkinkan Anda untuk mendefinisikan dan menggunakan custom font (font khusus) yang tidak tersedia di sistem pengguna. Dengan @font-face, Anda dapat mengunduh font dari server dan menampilkannya di website Anda.

Keuntungan @font-face:

  • âœ… Menggunakan font unik yang tidak tersedia di sistem pengguna
  • âœ… Membuat website lebih menarik dan profesional
  • âœ… Konsistensi tampilan font di semua browser
  • âœ… Meningkatkan branding website

Sintaks Dasar @font-face:

@font-face { font-family: 'Nama-Font'; src: url('path/ke/font.ttf') format('truetype'); font-weight: normal; font-style: normal; }

Penjelasan Properti:

1. font-family: Nama custom untuk font yang akan digunakan di CSS

2. src: Lokasi file font (lokal atau URL eksternal) dan formatnya

3. font-weight: Ketebalan font (normal, bold, 100-900)

4. font-style: Gaya font (normal, italic, oblique)

Format Font yang Didukung:

  • TTF (TrueType) - format('truetype')
  • OTF (OpenType) - format('opentype')
  • WOFF - format('woff') - Recommended
  • WOFF2 - format('woff2') - Lebih ringan
  • EOT (Embedded OpenType) - format('embedded-opentype') - IE lama

Contoh Penggunaan 1: Font Lokal

@font-face { font-family: 'CustomFont'; src: url('fonts/my-custom-font.ttf') format('truetype'); } h1 { font-family: 'CustomFont', serif; }

Contoh Penggunaan 2: Dari Google Fonts

@font-face { font-family: 'Roboto'; src: url('https://fonts.gstatic.com/s/roboto/v29/KFOmCnqEu92Fr1Mu7GxKOzY.woff2') format('woff2'); } body { font-family: 'Roboto', sans-serif; }

Contoh Penggunaan 3: Multiple Weights

@font-face { font-family: 'MyFont'; src: url('fonts/myfont-regular.woff2') format('woff2'); font-weight: 400; } @font-face { font-family: 'MyFont'; src: url('fonts/myfont-bold.woff2') format('woff2'); font-weight: 700; } .regular { font-family: 'MyFont', sans-serif; font-weight: 400; } .bold { font-family: 'MyFont', sans-serif; font-weight: 700; }

Praktik Terbaik:

ðŸ’¡ Tips:

  • Gunakan format WOFF2 untuk ukuran file lebih kecil
  • Sediakan fallback font (serif, sans-serif, monospace)
  • Pertahankan load time dengan tidak menggunakan terlalu banyak font
  • Gunakan font-display untuk kontrol loading behavior
  • Optimalkan ukuran file font untuk performa

Contoh Lengkap dengan font-display:

@font-face { font-family: 'ModernFont'; src: url('fonts/modern.woff2') format('woff2'), url('fonts/modern.woff') format('woff'); font-weight: 400; font-style: normal; font-display: swap; } /* font-display values: - auto: default browser behavior - block: text tidak terlihat sampai font loaded - swap: gunakan fallback dulu, ganti saat font loaded - fallback: timeout 100ms, gunakan fallback jika belum loaded - optional: jika font belum loaded dalam 100ms, tidak perlu mengganti */

Ringkasan:

@font-face adalah fitur CSS yang powerful untuk menggunakan custom font di website. Dengan syntax yang sederhana, Anda dapat membuat website lebih menarik dan sesuai dengan branding yang diinginkan. Selalu gunakan format font yang optimal dan sediakan fallback font untuk kompatibilitas maksimal.
