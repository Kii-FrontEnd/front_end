Background clip

CSS background-clip properti

CSS properti background-clip mendefinisikan seberapa jauh latar belakang (warna, gambar, atau gradien) harus meluas ke dalam sebuah elemen.
Properti ini dapat memiliki salah satu nilai berikut:

  • padding-box - latar belakang memanjang di belakang perbatasan. Ini adalah default
  • border-box - Latar belakang meluas ke tepi dalam perbatasan.
  • content-box - Latar belakang meluas ke tepi kotak konten.
