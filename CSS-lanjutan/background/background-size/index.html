properti background-size CSS

Properti background-size memungkinkan anda untuk menentukan ukuran gambar latar belakang.

background-size dapat ditentukan dalam panjang, persentase, atau dengan menggunakan salah satu kata kunci: auto, contain, atau cover.

background-size - auto, contain dan cover

Nilai kata kunci untuk background-size adalah auto, contain, dan cover.

  • Nilai auto adalah nilai default, dan menampilkan gambar latar belakang dalam ukuran aslinya.
  • Nilai contain adalah nilai dengan skala gambar atas atau bawah menyesuaikan ukuran area konten.
  • Nilai cover adalah nilai skala gambar menjadi sebesar mungkin sehingga area konten benar-benar tertutup oleh gambar latar belakang (lebar dan tingginya akan sama dengan, atau lebih besar dari area konten).
