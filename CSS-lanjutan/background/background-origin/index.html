CSS background-origin properti

Properti CSS background-origin menentukan posisi backgroung image. Properti ini dapat memiliki salah satu nilai berikut:

  • padding-box - Gambar latar belakang dimulai dari sudut kiri atas tepi padding. ini adalah default
  • border-box - Gambar latar belakang dimulai dari sudut kiri atas sisi border.
  • content-box - Gambar latar belakang dimulai dari sudut kiri atas sisi konten.
Catatan: Properti ini tidak berpengaruh jika background-attachment properti diatur ke "fixed".
