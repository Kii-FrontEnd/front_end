CSS Transisi

Definisi CSS Transisi:
CSS Transisi adalah fitur yang memungkinkan perubahan nilai properti CSS terjadi secara bertahap dan halus selama periode waktu tertentu, alih-alih berubah secara instan. Transisi menciptakan efek animasi yang menarik tanpa memerlukan JavaScript.

Properti-Properti CSS Transisi:

  • transition-property: Menentukan properti CSS mana yang akan dianimasikan (contoh: width, color, opacity)
  • transition-duration: Menentukan berapa lama transisi berlangsung (dalam detik atau milidetik, contoh: 2s, 500ms)
  • transition-timing-function: Menentukan kecepatan transisi selama durasi (ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier)
  • transition-delay: Menentukan penundaan sebelum transisi dimulai (dalam detik atau milidetik, contoh: 1s, 200ms)
  • transition: Shorthand untuk menggabungkan semua properti transisi di atas

Contoh transisi CSS

Contoh berikut menunjukkan elemen 100px * 100px pada <class: cth1;> dan <class: cth2;>. Yang <class: cth1;> menentukan efek transisi width 3s, sedangkan yang <class: cth2;> menentukan width, height dan background-color transisi.
