CSS 3D transformasi

Properti CSS transform menerapkan transformasi 2D atau 3D ke elemen. Properti ini memungkin anda untuk rotate, scale, translate, skew elemen.

CSS 3D mengubah fungsi

Dengan CSS transform properti yang dapat anda gunakan untuk fungsi transformasi 3D sebagai berikut:

  • rotateX()
  • rotateY()
  • rotateZ()

fungsi CSS rotateX()

Fungsi rotateX() memutar elemen di sekitar sumbu X (horizontal). Dengan menggunakan rotateX, elemen akan berputar secara vertikal, menciptakan efek 3D yang membuat elemen terlihat seperti sedang diputar dari atas ke bawah atau sebaliknya. Nilai rotasi diberikan dalam derajat (deg), di mana nilai positif memutar elemen ke arah depan dan nilai negatif memutar ke arah belakang.

fungsi CSS rotateY()

Fungsi rotateY() memutar elemen di sekitar sumbu Y (vertikal). Dengan menggunakan rotateY, elemen akan berputar secara horizontal, menciptakan efek 3D yang membuat elemen terlihat seperti sedang diputar dari kiri ke kanan atau sebaliknya. Nilai rotasi diberikan dalam derajat (deg), di mana nilai positif memutar elemen ke arah kanan dan nilai negatif memutar ke arah kiri.

fungsi CSS rotateZ()

Fungsi rotateZ() memutar elemen di sekitar sumbu Z (kedalaman). Dengan menggunakan rotateZ, elemen akan berputar secara melingkar pada bidang datar, menciptakan efek rotasi yang sama seperti rotasi 2D tradisional. Nilai rotasi diberikan dalam derajat (deg), di mana nilai positif memutar elemen searah jarum jam dan nilai negatif memutar berlawanan arah jarum jam.
