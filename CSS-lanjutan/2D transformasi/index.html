CSS 2d Transformasi

Properti CSS transform menerapkan transformasi 2d atau 3d ke elemen. Properti ini memungkinkan anda untuk translate, skala, rotate, dan skew elemen.

Jenis-jenis 2D Transformasi:

1. Translate

Menggeser elemen dari posisi aslinya

  • translate(x, y) - Geser horizontal dan vertikal
  • translateX(x) - Geser horizontal
  • translateY(y) - Geser vertikal
  • Nilai: pixel (px), persentase (%), atau nilai lainnya

2. Rotate

Memutar elemen dengan sudut tertentu

  • rotate(angle) - Putar searah jarum jam
  • Nilai: derajat (deg), radian (rad)
  • Contoh: rotate(45deg), rotate(-30deg)

3. Scale

Mengubah ukuran elemen

  • scale(x, y) - Skala horizontal dan vertikal
  • scaleX(x) - Skala horizontal
  • scaleY(y) - Skala vertikal
  • Nilai: angka (1 = normal, 0.5 = setengah, 2 = dua kali)

4. Skew

Memiringkan elemen

  • skew(x, y) - Miringkan horizontal dan vertikal
  • skewX(angle) - Miringkan horizontal
  • skewY(angle) - Miringkan vertikal
  • Nilai: derajat (deg), contoh: skew(20deg, 10deg)

5. Matrix

Kombinasi semua transformasi 2D dalam satu fungsi

  • matrix(a, b, c, d, e, f) - Transformasi kompleks
  • Nilai: angka-angka untuk matriks transformasi

CSS fungsi matrix()

Fungsi matrix() menggabungkan semua transformasi 2D (translate, rotate, scale, dan skew) dalam satu fungsi. Fungsi ini menggunakan enam parameter yang mewakili elemen-elemen dalam matriks transformasi.
Sintaks: matrix(a, b, c, d, e, f)

  • a - Skala horizontal (scaleX)
  • b - Miringkan horizontal (skewX)
  • c - Miringkan vertikal (skewY)
  • d - Skala vertikal (scaleY)
  • e - Geser horizontal (translateX)
  • f - Geser vertikal (translateY)
