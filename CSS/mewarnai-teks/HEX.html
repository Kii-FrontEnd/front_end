Nilai HEX

Warna heksadesimal ditentukan dengan: #RRGGBB(red, green, blue).
Dalam CSS, warna dapat ditentukan menggunakan nilai heksadesimal Dalam bentuk:

  • 6 digit: #rrggbb
  • 3 digit: #rgb

# rrggbb

Dimana rr(merah), gg(hijau), bb(biru) adalah nilai heksadesimal antara 00 dan ff (sama dengan desimal 0-255).
Misalnya, #ff0000 ditampilkan sebagai merah, karena merah diatur ke nilai tertinggi(ff) dan yang lainnya diatur dengan nilai 00(terendah).
Untuk menampilkan hitam, tetapkan semua nilai menjadi 00, seperti ini: #000000
untuk menampilkan putih, tetapkan warna nilai tertinggi ff, seperti ini: #ffffff
contoh #rrggbb:

#ff0000
#0000ff
#3cb371
#ee82ee
#ffa500
#6a5acd

Nuansa abu-abu sering didefinisikan menggunakan nilai yang sama untuk semua 3 sumber cahaya:

#3c3c3c
#616161
#787878
#b4b4b4
#f0f0f0
#f9f9f9

3 Digit Nilai HEX

Kode HEX 3 digit adalah singkatan untuk beberapa kode hex 6 digit.
Contohnya sebagai berikut:

# rgb

Dimana r, g, b mewakili komponen merah, hijau, dan biru dengan nilai antara 0 dan f.
Kode hekss 3 digit hanya dapat digunakan ketika kedua nilai (RR, GG dan BB) adalah sama untuk setiap komponen. Jadi, jika kita memiliki #ff00cc, itu bisa ditulis seperti ini: #f0c
Berikut contohnya:

  • #fc9 = #ffcc99
  • #f0f = #ff00ff
  • #b58 = #bb5588
