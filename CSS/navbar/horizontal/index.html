Dalam bilah navigasi horizontal, tautan navigasi ditumpuk secara horizontal (dari kiri ke kanan) di bagian atas halaman web. Ini adalah tata letak yang umum digunakan untuk bilah navigasi karena memanfaatkan ruang horizontal yang luas di bagian atas halaman web.

Dasar-dasar bar navigasi hrizontal adalah daftar unordead ul, dengan item daftar li untuk setiap tautan navigasi.
Hal ini juga umum untuk menambahkan elemen nav di sekitar elemen ul, yang berfungsi sebagai wadah untuk bar navigasi anda.

horizontal navbar menggunakan float

Salah satu cara untuk membuat bilah navigasi horizontal adalah dengan menggunakan properti CSS float untuk elemen li.
Di sini kita akan membuat bar navigasi horizontal dasar dengan warna latar belakang gelap dan mengubah warna latar belakang link ketika penggunan mengarahkan cursor diatas teks navbar:

horizontal navbar menggunakan flex

Cara lain untuk membuat bilah navigasi horizontal adalah dengan menggunakan properti CSS flexbox. Flexbox memberikan cara yang lebih fleksibel dan kuat untuk mengatur elemen dalam bar navigasi.
