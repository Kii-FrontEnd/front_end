CSS !important

!important aturan digunakan untuk memnberikan nilai properti tertentu adalah prioritas tertinggi, mengesampingkan aturan lain yang mungkin bertentangan.
aturan !important akan mengesampingkan SEMUA aturan CSS lainnya, termasuk aturan inline. Cara penulisan adalah dengan menambahkan !important setelah nilai properti, diikuti dengan titik koma (;).

Sintaks: 


    p {
        color: blue !important;
    }

beberapa kegunaan !important

!important digunakan untuk memaksa aturan CSS tertentu untuk diterapkan, meskipun aturan lain memiliki prioritas lebih tinggi. Sepert:

  1. Untuk mensponsori style yang tidak bisa dikesampingkan dengan cara lain. Ini bisa berguna ketika bekerja dengan CSS pihak ketiga atau framework di mana Anda tidak memiliki kontrol penuh atas aturan CSS yang ada. Dengan menggunakan !important, Anda dapat memastikan bahwa aturan tertentu diterapkan sesuai keinginan Anda.
  2. Untuk menghormati preferensi pengguna. Beberapa pengguna mungkin memiliki preferensi khusus untuk tampilan situs web, seperti mode gelap atau ukuran font yang lebih besar, dan sedikit animasi. CSS memiliki @media fiitur yang disebut prefers-reduced-motion yang memungkinkan pengembang web untuk mendeteksi preferensi ini dan menyesuaikan gaya situs web sesuai kebutuhan. Dalam kasus ini, menggunakan !important dapat membantu memastikan bahwa gaya yang sesuai diterapkan untuk menghormati preferensi pengguna.

  3. Untuk menciptakan style yang sangat spesifik dan tidak dapat diubah untuk yang spesifik elemen. Asumsikan kami ingin tampilan khusus untuk semua tombol link pada halaman.
