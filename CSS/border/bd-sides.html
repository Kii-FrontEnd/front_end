BORDER COLOR

Dari contoh-contoh pada halaman sebelumnya, anda telah melihat bahwa adalah mungkin untuk menentukan perbatasan yang berebda untuk masing-masing pihak.
Dalam CSS ada juga preperti untuk menentukan masing-masing perbatasan (atas, kanan, bawah, kiri).

  • border-top-style
  • border-right-style
  • border-bottom-style
  • border-left-style

Dalam contoh di bawah ini kita menggunakan empat sifat di atas untuk mengatur gaya setipa sisi perbatasan:
Contoh:

2 gaya border yang berbeda

Jadi di sini adalah cara kerjanya:

  1. Jika memakai border-style properti memiliki empat nilai:
    Gaya perbatasan: putus-putus double dashed yang solid:
    • nilai yang pertama adalah putus-putus
    • nilai kedua (kanan)adalah double
    • nilai ketiga (bawah) adalah dashed
    • nilai keempat (kiri) adalah solid
  2. Jika memiliki 3 nilai dengan border-style: putus-putus solid ganda
    • sisi pertama (atas) adalah putus-putus
    • sisi kedua (kanan dan kiri) adalah solid
    • sisi ketiga (bawah) adalah ganda
  3. jika border-style properti memiliki dua nilai dengan nilai: border-style: dotted padat
    • nilai pertama (atas dan bawah) adalah putus-putus
    • nilai kedua (kanan dan kiri) adalahh solid
  4. jika border-style properti memiliki satu nilai: border-style: putus-putus (dotted)
    • keempat sisi tersebut memiliki nilai dotted
