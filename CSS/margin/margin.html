MARGIN

Sifat margin CSS digunakan untuk menciptakan ruang di sekitar elemen, di luar perbatasan yang ditentukan.
Margin menentukan jarak antara perbatasan elemen dan sekitarnya.
Dengan CSS, anda memiliki kontrol penuh atas margin. CSS memiliki properti untuk menentapkan margin untuk setiap sisi individu dari elemen (atas, kanan, bawah, kiri), dan properti singkatan untuk pengaturan semua properti margin dalam satu deklarasi.

CSS memiliki properti untuk menentukan margin untuk masing-masing sisi dari elemen:

  • margin-top - menetapkan margin atas
  • margin-right - mengatur margin kanan
  • margin-bottom - mengatur margin bawah
  • margin-left - mengatur margin kiri

Semua properti margin dapat memiliki nilai-nilai berikut:

  • auto - browser menghitung margin
  • panjang - menentukan margin dalam px, pt, cm, dll
  • % - menentukan margin dalam % dari lebar elemen yang mengandung
  • mewarisi - menentukan bahwa margin harus diwarisi dari elemen induk
Tip: Nilai-nilai negatif juga diperbolehkan
