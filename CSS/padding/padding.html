PADDING

Sifat padding CSS digunakan untuk menghasilkan ruang di sekitar konten elemen, di dalam batas yang ditentukan.

Dengan CSS, anda memiliki kontrol penuh atas padding. Ada properti untuk pengaturan bantalan untuk setiap sisi elemen (atas, kanan, bawah, kiri), dan properti singkatan untuk pengaturan semua properti padding dalam satu deklarasi.

Padding - sisi individu

CSS memiliki properti untuk menentukan padding untuk setiap sisi elemen:

  • padding-top - mengatur padding atas
  • padding-right - mengatur padding kanan
  • padding-bottom - mengatur padding bawah
  • padding-left - mengatur padding kiri

Semua properti padding dapat memiliki nilai-nilai berikut:

  • panjang - menentukan margin dalam px, pt, cm, dll
  • % - menentukan margin dalam % dari lebar elemen yang mengandung
  • mewarisi - menentukan bahwa margin harus diwarisi dari elemen induk
Tip: Nilai-nilai negatif tidak diperbolehkan

PADDING DAN BOX SIZING

Properti Box Sizing didefinisikan bagaimana lebar dan tinggi elemen dihitung: harus mencangkup padding dan border, atau tidak.
Properti ini memiliki nilai-nilai berikut:

  • Content-box - ini adalah default. Sifat lebar dan tinggi hanya mencangkup konten (border dan padding tidak termasuk)
  • border-box - Sifat lebar dan tinggi termasuk konten, padding, dan border.

Jadi, untuk menjaga lebar pada 300px, tidak peduli jumlah padding, anda dapat menggunakan box-sizing: border-box;. Hal ini menyebabkan elemen memperahankan lebar sebenarnya; jika anda meningkatkan padding, ruang konten yang tersedia akan berkurang.
