OUTLINE COLOR

Properti outline-color digunakan untuk mengatur warna garis besar. Warna dapat diatur oleh:

  • Nama - menentukan nama warna, seperti "merah"
  • HEX - menentukan nilai hex, seperti "#ff0000"
  • RGB - tentukan nilai RGB, seperti "rgb (255,0,0)"
  • HSL - tentukan nilai HSL, seperti "hsl(0, 100%, 50%)"
  • invert - melakukan inversi warna (yang memastikan bahwa garis besar terlihat, terlepas dari latar belakang warna)
