OUTLINE SHORTHAND

Properti outline adalah properti singkatan untuk menetapkan masing-masing bagian outline berikut:

  • outline width
  • outline style
  • outline color

Anda dapat menentukan satu, dua, atau tiga nilai dari daftar di atas. Urutan nilai tidak penting.

CSS Outline Contoh Shorthand

Berikut adalah beberapa outline yang berbeda yang ditentukan dengan singkatan outline properti:

sebuah outline dashed
sebuah outline dotted red
sebuah outline 7px yellow
sebuah outline thick pink

CSS Outline Dengan Corner Bulat

Garis besar juga dapat memiliki sudut bulat. Properti border-radius adalah digunakan untuk menambahkan batas bulat ke elemen.
Berikut adalah beberapa garis besar yang berbeda dengan sudut bulat:

sebuah outline dashed
sebuah outline dotted red
sebuah outline 7px yellow
sebuah outline thick pink
