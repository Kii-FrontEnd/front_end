OUTLINE WIDTH

Properti outline-width menentukan lebar garis besar, dan dapat memiliki salah satu nilai berikut:

  • thin(biasanyya 1px)
  • medium(biasanya 3px)
  • thick(biasanya 5px)
  • Ukuran tertentu (dalam px, pt, cm, em, dll)
