styling bentuk form

CSS digunakan untuk gaya bentuk HTML. Tampilan bentuk HTML dapat diubah menggunakan CSS dengan menargetkan elemen formulir seperti <input>, <label>, <select>, <button> dan lainnya.




styling bentuk input fields

Dengan CSS anda dapat menata sebagian besar jenis input yang berbeda, seperti text, password, checkbox, radio, submit, dan lainnya. Anda juga bisa mengubah input dan form button.
Beberapa properti CSS yang umum digunakan untuk menata input fields adalah:

  • width dan height: Mengatur ukuran input fields.
  • padding dan margin: Menambahkan ruang di dalam dan di luar input fields.
  • border: Mengubah gaya, warna, dan lebar border input fields.
  • background-color: Mengubah warna latar belakang input fields.
  • font-size dan color: Mengubah ukuran dan warna teks di dalam input fields.
  • :focus pseudo-class: Menata input fields saat mereka mendapatkan fokus.
