BOX MODEL

Dalam CSS, istilah "Box Model" digunakan saat berbicara tentang desain web dan tata letak.
Model CSS box pada dasarnya adalah kotak yang membungkus setiap elemen HTML.
Setiap kotak terdiri dari empat bagian: konten, padding, border dan margin. Gambar dibawah ini menggambarkan model kotak CSS:

  • konten - Isi kotak, dimana tes dan gambar muncul
  • padding - Membersihkan area di sekitar konten. padding adalah transparan
  • border - border yang melintas padding dan konten
  • margin - Membersihkan area di luar border. Margin adalah transparan

Box Model memungkinkan kita untuk menambahkan batas di sekitar elemen, dan untuk menentukan ruang antara elemen.
Contoh:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Velit mollitia architecto quo. Voluptates doloremque cupiditate ipsa velit, animi aut quae iusto molestias sed, autem dicta.

Width dan Height Elemen

untuk mengatur width dan height elemen dengan benar di semua browser, anda perlu tahu cara kerja model kotak.

Penting: Ketika anda mengatur sifat lebar dan tinggi dari elemen dnegan CSS, anda hanya mengatur lebar dan tinggi are konten. Untuk menghitung total lebar dan tinggi elemen, anda juga harus menyertakan padding dan border.

Contoh:
Class two ini akan memiliki lebar total 520px dan tinggi total dari 80px:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Eaque, ipsam. Ducimus enim blanditiis ea recusandae nisi magni, nesciunt omnis fugiat, minima, maxime quisquam cupiditate asperiores illo nihil! Sunt, vero adipisci!

Beriikut adalah Perhitungannya:

320px (width area konten) + 40px (padding left + padding Right) + 10px (border left + border right) = 370px (lebar total)

100px (height area konten) + 40px (padding left + padding Right) + 10px (border left + border right) = 150px (tinggi lebar)
