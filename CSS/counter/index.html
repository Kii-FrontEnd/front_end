CSS Perhitungan/counter

Dengan menggunakan CSS Counter, Anda dapat menambahkan penomoran otomatis pada elemen HTML seperti daftar atau paragraf. Ini sangat berguna untuk membuat daftar bernomor atau menandai bagian tertentu dalam dokumen Anda tanpa perlu menambahkan nomor secara manual.

CSS automatic numbering dengan counter

Perhitungan CSS seperti "variabel". Nilai variabel dapat ditingkatkan atau diatur ulang menggunakan aturan CSS. Anda dapat menggunakan perhitungan CSS untuk membuat penomoran otomatis pada elemen HTML. Berikut properti yang digunakan untuk CSS counter:

  • counter-reset - mengatur ulang nilai counter ke nilai tertentu.
  • counter-increment - meningkatkan nilai counter sebesar nilai tertentu.
  • content- - menyisipkan nilai counter ke dalam konten elemen. (Digunakan bersama dengan pseudo-elemen seperti ::before atau ::after, untuk memasukkan nilai counter ke dalam konten elemen.)
  • counter() dan counters() - fungsi yang digunakan untuk mengambil nilai counter.

Untuk menggunakan counter CSS, pertama-tama harus diuat dengan counter-reset properti.

counter CSS() fungsi

Fungsi counter() digunakan untuk mengambil nilai counter tunggal dan menampilkannya di dalam elemen. Fungsi ini biasanya digunakan bersama dengan pseudo-elemen seperti ::before atau ::after.
Berikut adalah sintaks dasar untuk fungsi counter(): 

                counters(counter-name, style)

Di mana counter-name adalah nama counter yang ingin Anda ambil nilainya, dan style adalah gaya penomoran yang diinginkan (misalnya, decimal, lower-roman, upper-roman, lower-alpha, upper-alpha, dll.).
