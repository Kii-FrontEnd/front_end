pseudo element

Sebuah CSS pseudo-element adalah kata kunci yang dapat ditambahkan ke selector, pada gaya bagian tertentu dari suatu elemen.
Beberapa pengguanan umum untuk pseudo-element:

  • Gaya huruf pertama dan baris pertama dari elemen
  • Masukkan content sebelum atau sesudah elemen
  • Gaya penanda item daftar
  • Gaya bagian yang dipilih pengguna dari suatu elemen
  • Gaya viewbox di balik kotak dialog

Sintaks

pseudo-element dilambangkan dengan petik ganda (::) diikuti oleh nama pseudo-element:

selector::pseudo-element-name {
CSS properties
}

CSS ::first-line Pseudo-element

::first-line pseudo-element digunakan untuk menambahkan gaya khusus ke baris pertama dari sebuah teks.
Catatan: ::first-line pseudo-element hanya dapat diterapkan pada tingkat block elemen.

Contoh:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Error veniam quia voluptatem aspernatur necessitatibus dolorum beatae laudantium eius aliquam aperiam. Illum molestias fugiat, quae autem quisquam porro consectetur dolor reprehenderit.

CSS ::first-letter Pseudo-element

::first-letter pseudo-element digunakan untuk menambahkan gaya khusus ke huruf pertama dari sebuah teks.
Catatan: ::first-letter pseudo-element hanya dapat diterapkan pada tingkat block elemen.

Contoh:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Error veniam quia voluptatem aspernatur necessitatibus dolorum beatae laudantium eius aliquam aperiam. Illum molestias fugiat, quae autem quisquam porro consectetur dolor reprehenderit.

CSS ::before pseudo-element

::before pseudo-element digunakan untuk memasukkan beberapa konten sebelum konten dari elemen tertentu.
Gunakan content properti untuk menentukan konten yang akan dimasukkan.

Contoh

Memasukkan gambar sebelum isi dari content:

Welcome Guys

CSS ::after pseudo-element

::after pseudo-element digunakan untuk memasukkan beberapa konten sesudah konten dari elemen tertentu.
Gunakan content properti untuk menentukan konten yang akan dimasukkan.

Contoh

Memasukkan gambar sesudah isi dari content:

Welcome Guys

CSS ::marker pseudo-element

pseudo-element ::marker digunakan untuk menandakan item pada daftar.

Contoh

pseudo-element akan menandakan seluruh body page yang menggunakan elemen list (ul,ul, li) pada html.
Gaya penanda item daftar:

  • Gym
  • Workout
  • Music
  1. Gitar
  2. Dumble
  3. Barbel

CSS ::selection pseudo-element

::selection pseudo-element digunakan untuk style bagian dari teks yang dipilih oleh developer.

Contoh

Gaya teks yang dipilih pengguna dengan warnah merah, dan latar belakang kuning:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Neque incidunt dolorem eius exercitationem tenetur eaque, alias soluta ipsam tempore repellendus.
