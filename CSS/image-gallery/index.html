image gallery

galeri gambar CSS adalah kumpulan gambar yang ditampilkan dengan cara yang teratur, dan sering responsive, di halaman web. Galeri ini biasanya menggunakan CSS untuk mengatur tata letak, gaya, dan interaktivitas gambar.

struktur HTML untuk galeri gambar adalah:

  • elemen wadah untuk membungkus seluruh galeri (seperti div dengan kelas gallery).
  • elemen individual untuk setiap item galeri (seperti div dengan kelas gallery-item).
