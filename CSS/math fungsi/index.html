CSS Matematika Fungsi

Fungsi matematika CSS memungkinkan eksprei matematika digunakan sebagai nilai properti. Dalam bab ini, kita akan menjelaskan fungsi matematika berikut:

  • calc()
  • max()
  • min()
  • clamp()

Fungsi dari CSS Calc()

Fungsi calc() melakukan perhitungan matemis yang akan digunakan sebagai nilai properti.

Fungsi calc() mendukung penambahan (+), pengurangan(-), perkalian(*), dan pembagian(/), serta dapat menggabungkan beberapa unit, seperti piksel(px) dan persentase(%).

CSS syntax

calc(expression)

Contoh

Gunakan calc() untuk menghitung width dan height elemen <div>

Fungsi calc()

width #calc diatur ke 100% dikurangi 100px untuk padding. Hal ini memastikan bahwa #calc mengambil 100% lebar wadah, dengan offset 50px di setiap sisi (dengan margin: auto;).
height #calc diatur ke 30% dari tinggi viewport + 50px.

Fungsi calc()

Fungsi CSS max()

Fungsi max() megambil daftar nilai yang dipisahkan koma, dan menggunakan nilai terbesar dari daftar sebagai nilai properti.

max(value1, value2, ....)

CSS syntax

Contoh

Gunakan max() untuk mengatur width #max ke nilai mana yang terbesar, antara 50% atau 300px.

Fungsi max()

Gunakan max() untuk mengatur lebar #max ke mana saja yang terbesar: 50% atau 300px. Ubah ukuran jendela browser untuk melihat efeknya!

Fungsi max()

Fungsi CSS min()

Fungsi min() megambil daftar nilai yang dipisahkan koma, dan menggunakan nilai terkecil dari daftar sebagai nilai properti.

CSS syntax

min(value1, value2, ....)

Contoh

Gunakan min() untuk mengatur width #max ke nilai mana yang terbesar, antara 50% atau 300px.

Fungsi min()

Gunakan min() untuk mengatur lebar #max ke mana saja yang terbesar: 50% atau 300px. Ubah ukuran jendela browser untuk melihat efeknya!

Fungsi min()

CSS fungsi clamp()

Fungsi clamp() digunakan untuk menetapkan nilai yang akan menyesuaikan secara responsif antara nilai minimum dan nilai maksimum tergantung pada ukuran viewport.

Fungsi clamp() memiliki 3 paramater: niali minimum, parafase nilai dan nilai maksimum. Jika nilai yang disukai lebih kecil dari minimum nilai atau lebih besar dari nilai maksimum, nilai yang disukai digunakan.

CSS syntax

clamp(min, preferred, max)

contoh
