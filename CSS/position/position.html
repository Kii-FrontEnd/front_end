POSITION

Position adalah mengatur penempatan elemen dalam halaman web.
Dengan position CSS, anda dapat mengesampingkan aliran dokumen default. Properti ini dapat memiliki salah satu nilai darikut:

  • static - ini adalah nilai default
  • relativ
  • fixed
  • absolute
  • sticky

Elemen kemudian diposisikan ke lokasi akhir mereka dengan top, bottom, left dan right properti.

Position Static

Semua elemen HTML diposisikan static secara default. Elemene ini tidak terpengaruh oleh properti atas, bawah, kiri, kanan.
Sebuah elemen dengan position: static; selalu diposisikan sesuai dengan aliran normal halaman:

elemen ini memiliki posisi: static;

Posisi Relative

Sebuah elemen dengan position: relative; diposisikan relatif terhadap posisi normalnya dalam aliran dokumen.
Menetapkan properti atas, kanan, bawah, dan kiri akan menyebabkan elemen yang akan disesuaikan jauh dari posisi normalnya. Konten lain tidak akan disesuaikan agar ssuai dengan celah apa pun yang ditinggalkan oleh elemen.

properti ini memiliki posisi relative, dan miring 30px ke kanan posisi normalnya.

Posisi Fixed

Sebuah elemen dengan position: fixed; diposisikan relatif terhadap viewport, yang berarti selalu Tetap di tempat yang sama bahkan jika halaman itu digulir. Bagian atas, sifat kanan, bawah, dan kiri digunakan untuk mengatur lokasi akhir elemen.
Elemen tetap tidak meninggalkan celah di halaman di mana biasanya berada. Perhatikan elemen tetap di sudut kanan bawah halaman. Berikut Contohnya:

elemen ini memiliki position: fixed;

Posisi absolute

Sebuah elemen dengan position: absolute; diposisikan relatif terhadap ancestors/nenek moyang terdekat (dengan posisi lain selain static). Namun; jika elemen posisi absolute tidak memiliki ancestors/nenek moyang yang diposisikan, itu menggunakan tubuh dokumen, dan bergerak bersama dengan halaman bergulir.

Catatan: Elemen yang diposisikan absolut dihapus dari normal dokumen aliran, dan dapat tumpang tindih dengan elemen lainnya. Berikut contohnya;

Elemen ini memiliki posisi relative
elemen ini memiliki posisi absolute

Posisi Sticky

Elemen dengan position: sticky; beralih antara elemen relative dan fixed posisi, tergantung pada posisi gulir.
Unsur lengket diposisikan relaif sampai posisi gulir tertentu mencapai - kemudian "sticky" di tempat itu (seperti posisi fixed).

Catatan Anda harus menentukan setidaknya salah satu dari top, right, bottom dan left properti, agar posisi sticky aktif

Coba Scroll kebawah frame ini untuk melihat apakah posisi sticky aktif
