dropdown

CSS dropdown digunakan untuk menampilkan daftar opsi atau konten saat pengguna klik atau hover dia atas elemen, seperti tombol atau tautan navigasi.

Sebuah penurunan CSS terdiri dari elemen pemicu (seperti div, button, p, a, danlainnya.).
Ketika elemen pemicu diklik atau hover di atas konten, maka opsi dropdown akan ditampilkan.
Konten dropdown adalah elemen kontainer (misalnya div) yang menyimpaan konten tersembunyi (bisa berupa teks, link, gambar, dll.).

penjelasan contoh diatas

  • kelas dropdown menggunakan position: relative;, yang diperlukan ketika kita ingin konten dropdown ditempatkan tepat di bawah elemen pemicu (dropdown konten akan digunakan position: abbsolute; ).
  • kelas dropdown-content memegang konten dropdown. ha ini disembunyikan oleh default, dan akan ditampalikan pada saat peggunan menghover teks.
  • elemen box-shadow digunakan untuk membuat konten dropdown terlihat seperti kartu.
  • selector :hover digunakan untuk menunjukkan dropdown konten saat pengguna menghover mouse di atas elemen kelas dropdown
