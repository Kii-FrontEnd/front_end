BACKGROUND COLOR

Dalam CSS, warna yang paling sering ditentukan oleh:

  • nama warna valid - seperti "red, blue, orange, green, grey dan lainnya"
  • nilai HEX - seperti "ff0000"
  • nilai RGB - seperti "rgb(255, 0 , 0)"

untuk lebih jelas lihatlah CSS Color Values W3School untuk lengkap daftar nilai warna yang mungkin digunakan.
