CSS Sudut Bulat

CSS border-radius properti

Properti border-radius mendefinisikan radius dari sudut elemen. Properti ini dapat diterapkan untuk smeua elemen dengan background-color, border,background-image.

Berikut adalah tiga contohnya:

  1. Sudut bulat untuk elemen dengan background-color
    Sudut-sudut bulat!
  2. Sudut bulat untuk elemen dengan border
    Sudut-sudut bulat!
  3. Sudut bulat untuk elemen dengan background-image
    Sudut-sudut bulat!
Tips: Properti border-radius adalah singkatan dari border-top-left-radius, border-top-right-radius, border-bottom-right-radius dan border-bottom-left-radius.
