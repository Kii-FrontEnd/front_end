CSS Sudut Bulat

CSS border-radius properti

Properti border-radius mendefinisikan radius dari sudut elemen. Properti ini dapat diterapkan untuk smeua elemen dengan background-color, border,background-image.

Berikut adalah tiga contohnya:

  1. Sudut bulat untuk elemen dengan background-color
    Sudut-sudut bulat!
  2. Sudut bulat untuk elemen dengan border
    Sudut-sudut bulat!
  3. Sudut bulat untuk elemen dengan background-image
    Sudut-sudut bulat!
Tips: Properti border-radius adalah singkatan dari border-top-left-radius, border-top-right-radius, border-bottom-right-radius dan border-bottom-left-radius.

CSS border-radius - menentukan setiap sudut

Properti border-radius dapat memiliki satu elemen dari empat nilai. Berikut adalah aturannya:

empat nilai - border-radius: 20px 55px 35px 10px: (pertama nilai berlaku untuk sudut kiri atas, nilai kedua berlaku untuk sudut kanan atas, nilai ketiga berlaku untuk sudut kanan bawah, dan nilai keempat berlaku untuk sudut kiri bawah):

tiga nilai - border-radius: 20px 55px 35px: (pertama nilai untuk sudut kiri atas, kedua nilai untuk sudut kanan atas dan kiri bawah, ketiga nilai untuk sudut kanan bawah)

dua nilai - border-radius: 20px 55px: (pertama nilai untuk sudut kiri atas dan kanan bawah, kedua nilai untuk sudut kanan atas dan kiri bawah)

satu nilai - border-radius: 20px: (nilai berlaku untuk semua empat sudut)

CSS border-radius - Bentuk Elliptical dan Melingkar

Properti border-radius juga dapat digunakan untuk membuat bentuk elliptical (elips) dan melingkar (lingkaran). Anda dapat menggunakan nilai horizontal dan vertikal dipisahkan dengan garis miring (/).

Bentuk Melingkar - border-radius: 50%: Nilai 50% akan membuat elemen menjadi bentuk lingkaran sempurna jika elemen tersebut memiliki ukuran yang sama (width dan height).

Bentuk Elliptical - border-radius: 50% / 25%: Menggunakan format horizontal / vertikal untuk membuat bentuk elips.

Bentuk Elliptical - border-radius: 25% / 50%: Contoh lain dari bentuk elips dengan proporsi berbeda.

Bentuk Elliptical Custom - border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%: Kombinasi nilai yang lebih kompleks untuk bentuk organik yang menarik.
