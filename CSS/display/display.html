display

Properti display adalah properti CSS yang pentig untuk mengendalikan layout. Ini menentukan apakah elemen diperlukan sebagai inline atau block. Setiap elemen HTML memiliki nilai tampilan default, tergantung pada jenis elemen apa itu. Nilai tampilan default untuk sebagian besar elemen adalah block dan inline. display properti digunakan untuk mengubah prilaku tampilan default dari elemen HTML.

Elemen Block-level

Elemen block-level selalu dimulia pada baris baru dan mengambil lebar penuh yang tersedia (melebar ke kiri dan kanan sejauh yang bisa).
Contoh elemen tingkat block/block-level:

  • div
  • h1-h6
  • p
  • form
  • header
  • footer
  • section

Elemen Inline

Elemen inline tidak mulai pada baris baru dan hanya mengambil lebar sebanyak yang diperlukan. Ini adalah elemen inline span di dalam sebuah paragraf.
Contoh dari elemen inline:

  • span
  • a
  • img

Nilai tampilan umum

Value Description
inline Menampilkan elemen sebagai elemen inline
block Menampilkan elemen sebagai elemen block
contents Membuat container menghilang, membuat elemen turunannya menjadi anak elemen yang naik level berikutnya di DOM
flex Menampilkan elemen sebagai wadah fleksibel tingkat blok
grid Menampilkan elemen sebagai wadah grid tingkat blok
inline-block Menampilkan elemen sebagai wadah blok tingkat inline. Elemen itu sendiri diformat sebagai elemen inline, tetapi Anda dapat menerapkan nilai tinggi, lebar, padding, dan margin
none Elemen ini sepenuhnya tersembunyi dari aliran dokumen (tidak memakan ruang).

Tampilan CSS: none;

Saat menggunakan display: none; elemen benar-benar tersembunyi dari aliran dokumen dan tidak mengambil di atas setiap ruang. Hal ini biasa digunakan dengan javascript untuk menyembunyikan atau menunjukkan elemen tanpa menghapus dan Menambahkannya.

Menyembunyikan elemen menggunakan display none atau visibility hidden

Perbedaan antara display: none; dan visibilitas: hidden;

visibilitas: tersembunyi menyembunyikan elemen, tetapi masih memakan ruang di tata letak.
display:none menghapus elemen dari dokumen. Itu tidak memakan ruang apa pun.
