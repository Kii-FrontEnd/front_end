membuat link untuk menghubungkan halaman web
---------------------------------------------------------

link atau Hyperlink adalah elemen HTML yang digunakan untuk menghubungkan satu halaman web dengan halaman web lainnya. Link dapat mengarah ke halaman dalam situs yang sama atau ke situs eksternal. 
Untuk membuat link, kita menggunakan tag `<a>` dengan atribut `href` yang berisi alamat tujuan link tersebut. Contoh penggunaan link sederhana:

<a href="https://www.petanikode.com">Kunjungi Example.com</a>

jenis-jenis link pada HTML
-----
1. Link ke halaman lain dalam situs yang sama (internal link)
2. Link ke halaman luar atau situs eksternal (external link)
3. Link ke bagian tertentu dalam halaman yang sama (anchor link)
4. Link untuk mengirim email (mailto link)
5. Link untuk melakukan panggilan telepon (tel link)

- internal link adalah link yang mengarah ke halaman lain dalam situs yang sama. Contoh penggunaan internal link:

<a href="about.html">Tentang Kami</a>
<a href="contact.html">Kontak</a>

- external link adalah link yang mengarah ke halaman luar atau situs eksternal. Contoh penggunaan external link:

<a href="https://www.google.com">Google</a>
<a href="https://www.facebook.com">Facebook</a>

- anchor link adalah link yang mengarah ke bagian tertentu dalam halaman yang sama. Contoh penggunaan anchor link:

<a href="#section1">Ke Bagian 1</a>
<a href="#section2">Ke Bagian 2</a>

- mailto link adalah link yang digunakan untuk mengirim email. Contoh penggunaan mailto link:

<a href="mailto:example@example.com">Kirim Email</a>

- tel link  adalah link yang digunakan untuk melakukan panggilan telepon. Contoh penggunaan tel link:

<a href="tel:+6281234567890">Telepon</a>

ATRIBUT-ATRIBUT UNTUK Link

- `href`: Atribut ini digunakan untuk menentukan alamat tujuan link. Alamat tersebut bisa berupa URL lengkap, alamat relatif, atau bahkan ID elemen dalam halaman yang sama.
- `target`: Atribut ini digunakan untuk menentukan cara membuka link. Nilai yang umum digunakan adalah `_blank` untuk membuka link di tab baru, `_self` untuk membuka link di tab yang sama (default), `_parent` untuk membuka link di frame parent, dan `_top` untuk membuka link di seluruh jendela browser.
- `title`: Atribut ini digunakan untuk memberikan informasi tambahan tentang link. Informasi ini akan muncul sebagai tooltip saat pengguna mengarahkan kursor ke link.
- 'rel' : Atribut ini digunakan untuk menentukan hubungan antara dokumen saat ini dan dokumen yang ditautkan. Nilai yang umum digunakan adalah `nofollow` untuk mencegah mesin pencari mengikuti link tersebut, `noopener` untuk mencegah akses ke objek window baru, dan `noreferrer` untuk mencegah pengiriman informasi referer.
