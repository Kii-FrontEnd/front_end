Pengenalan Dasar HTML 
--------------------------------------

HTML(HyperText Markup Language) merupakan bahasa dasar untuk membuat sebuah web, penanda atau markup ini, nanti akan kita sebut dengan tag. HTML bukanlah sebuah bahasa pemrograman, 
melainkan sebuah markup language yang digunakan untuk membuat struktur halaman web. Dengan HTML, kita bisa menentukan bagaimana konten-konten di halaman web tersebut akan ditampilkan di browser.
HTML terdiri dari serangkaian elemen yang digunakan untuk membungkus konten agar dapat ditampilkan dengan cara tertentu. Setiap elemen HTML didefinisikan oleh tag pembuka dan tag penutup, 
dengan konten di antara keduanya. Sebagai contoh, tag `<p>` digunakan untuk membuat paragraf, dan tag `<h1>` hingga `<h6>` digunakan untuk membuat heading dengan tingkat hierarki yang berbeda.
Elemen-elemen HTML dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut. 
Contohnya, atribut `href` pada tag `<a>` digunakan untuk menentukan alamat tujuan dari sebuah tautan. Seiring perkembangan web, HTML terus diperbarui dengan versi-versi baru yang membawa fitur-fitur terbaru. Saat ini, 
HTML5 merupakan standar terkini yang digunakan dalam pengembangan web.

-Peralatan untuk belajar HTML
Untuk belajar HTML, kamu tidak memerlukan peralatan khusus. Kamu hanya membutuhkan:
1. Komputer atau laptop dengan sistem operasi apa saja (Windows, macOS, Linux).
2. Browser web seperti Google Chrome, Mozilla Firefox, Safari, atau Microsoft Edge untuk melihat hasil kode HTML yang dibuat.
3. Teks editor seperti Visual Studio Code, Sublime Text, atau Notepad++ untuk menulis kode HTML.

-Nama File Untuk HTML
Ada beberapa hal yang perlu diperhatikan dalam membuat nama file HTML:
1. Nama file harus diakhiri dengan ekstensi `.html`, contohnya `index.html` atau `about.html`.
2. Hindari penggunaan spasi dalam nama file, gunakan tanda hubung (-) atau garis bawah (_) sebagai pengganti spasi. Sebagai contoh, `halaman-utama.html` atau `halaman_utama.html`.
3. Gunakan huruf kecil untuk nama file agar konsisten dan menghindari masalah kompatibilitas di server yang case-sensitive. Sebagai contoh, `index.html` lebih disarankan daripada `Index.html`.

-Struktur Dasar HTML
Setiap dokumen HTML harus memiliki struktur dasar yang terdiri dari beberapa elemen penting. Struktur dasar ini mencakup deklarasi tipe dokumen, elemen `<html>`, `<head>`, dan `<body>`.
Berikut adalah contoh struktur dasar HTML:
<!DOCTYPE html>
<html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <h1>Selamat Datang di Website Saya</h1>
        <p>Ini adalah paragraf pertama.</p>
    </body>
</html>
Penjelasan dari struktur dasar HTML di atas:
- `<!DOCTYPE html>`: Mendeklarasikan bahwa dokumen ini adalah dokumen HTML5.
- `<html>`: Elemen root dari halaman HTML yang membungkus semua konten di halaman tersebut.
- `<head>`: Elemen yang berisi informasi meta tentang dokumen, seperti judul halaman yang ditampilkan di tab browser.
- `<title>`: Elemen yang menentukan judul halaman web yang akan ditampilkan di tab browser.
- `<body>`: Elemen yang berisi semua konten halaman web yang akan ditampilkan kepada pengguna, seperti teks, gambar, dan elemen lainnya.
